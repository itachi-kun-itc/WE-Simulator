# WE-Simulator

震源や断層、マグニチュード、深さなどの条件を設定し、地震の揺れと広がりを地図上で再現できる防災学習シミュレーターです。自由な条件でのシミュレーションに加え、過去の地震や想定地震の再現、結果の比較・共有にも対応しています。

▶ [WE-Simulatorを開く](https://itachi-kun-itc.github.io/WE-Simulator/)

## 主な機能

- **震源シミュレーション** — 震源位置、マグニチュード、深さを設定し、各地の揺れを地図上に表示
- **有限断層シミュレーション** — 矩形断層を複数の小震源に分け、破壊の進行や波の到達時間、断層からの距離を考慮して計算
- **多様な断層シナリオ** — 南海トラフ巨大地震、活断層、任意に設定した矩形断層に対応
- **過去・想定地震の再現** — 用意された地震データから条件を選んでシミュレーション
- **表示と比較** — 震度分布や各種情報を地図レイヤーで確認し、複数の結果を比較
- **シナリオの保存・共有** — 作成した条件を保存・読み込みし、ほかの利用者と共有
- **防災学習・コミュニティ機能** — シミュレーション結果を使った学習、投稿、通知、アカウント機能に対応

## 活用例

- 地震の規模や深さによる揺れ方の違いを比較する
- 断層の形状や破壊開始位置による影響を調べる
- 過去の地震や想定地震について学ぶ
- 作成したシナリオを共有し、防災について考える

詳しい使い方や各機能については、[WE-Simulator公式Wiki](https://github.com/itachi-kun-itc/WE-Simulator/wiki)をご覧ください。

> 本アプリの計算結果は防災学習を目的とした推定であり、実際の地震による被害や揺れを保証・予測するものではありません。
